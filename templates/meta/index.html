API Documentation

Endpoint: /verify

Method: GET

Description: Verifies the KYC information based on the provided address and at least one additional field.

Parameters:

  • address (string, required): Ethereum address of the user.
  • name (string, optional): Name of the user.
  • age (integer, optional): Age of the user.
  • location (string, optional): Location of the user.
  • id_number (string, optional): ID number of the user.

Note: At least one of the fields name, age, location, or id_number must be provided along with address.

Response:

The response is a JSON object with the following structure:

{
    "status": "success" | "error",
    "result": true | false
}

status: Returns success if the request is processed correctly, or error if there is a validation or server issue.

result: A boolean value indicating whether the verification was successful.
